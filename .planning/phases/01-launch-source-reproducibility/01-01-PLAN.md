| T-01-02 | Information Disclosure | frontend build inputs | medium | mitigate | Do not embed `.env` values or secret placeholders into `frontend/`; keep the frontend static and same-origin without client-side secret injection |
| T-01-03 | Repudiation | Docker build reproducibility | medium | mitigate | Keep all frontend dependencies and build scripts in version control via `frontend/package-lock.json` and deterministic `npm ci` |
</threat_model>
